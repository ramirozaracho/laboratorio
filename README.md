# Análisis de Visas H-1B (EE. UU.) desde el año 2015 hasta 2019

Este proyecto analiza datos de solicitudes de visas H-1B para entender las tendencias de aprobación, los empleadores más activos y las ciudades con mayores oportunidades.

---

# Descripción

El objetivo del análisis es identificar patrones relevantes en la concesión de visas H-1B en Estados Unidos entre distintos años y sectores.

El dataset contiene información como:
- **Año**
- **Empleador**
- **Ocupación**
- **Ciudad / Estado**
- **Salario previsto**
- **Estado de la solicitud (CERTIFIED, DENIED, etc.)**
